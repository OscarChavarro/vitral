Note that if this applet is not working, can be due to a different set of causes, mostly:
  • vsdk.jar library package must be copied (not linked) to the classes directory
  • In Unix/Linux operating systems, do not use symbolic links, as applet subsystem gives security exceptions
  • In windows based browsers, the JOGL library must be previously signed (to study better). One way to sign it is to test the examples in the JOGL page.
