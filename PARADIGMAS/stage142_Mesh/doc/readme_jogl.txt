This example needs the JOGL library.

Downloadable from:
  https://jogl.dev.java.net/
  For windows you need to download jogl.jar and jogl-natives-win32.jar

Install JOGL:
  Copy the 2 jar files into the main directory, 
  and unzip the 2 dll files from jogl-natives-win32.jar
  In windows: copy to jdk/jre/lib/ext

Compile
  Use compile.bat, note the use of classpath option

Run
  Enjoy using run.bat, note that the .dll files are needed in the
  running path of the java command or in the Windows“ system32 
  directory

NOTE: if running from JBuilder, you must place the dll files in the main directory
