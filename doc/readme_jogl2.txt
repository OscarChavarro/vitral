This example needs the JOGL library.

Downloadable from:
  https://jogl.dev.java.net/
  For windows you need to download jogl.jar and jogl-natives-win32.jar

Install JOGL:
  Copy the 2 jar files into:
   - jdk.../jre/lib/ext AND
   - jre.../lib/ext
  and unzip the 2 dll files from jogl-natives-win32.jar into system32 directory
