Vitral SDK applet examples

This page contains a list of applet examples based on the VitralSDK. Note that for each example there are two versions: one for environments with preinstalled JOGL library ("stand alone" version), and other for environments without JOGL library installed ("subapplet" version).

Vitral SDK simple example programs (toolkit level only)

The following programs are based on the Vitral SDK toolkit and JOGL. These group of examples are targetted to programmers, and shows basic functionality available in the VitralSDK toolkit. They are characterized by:
  • Common interaction, usually without swing elements, and common camera manipulation.
  • Small source code in just one class.
  • All these programs are based on CameraExample. Once CameraExample is studied, all programs will appear as additions to that example, so each program focuses in one simple "atomic" idea.

CameraExample: this is a basic VitralSDK example based on JOGL hello world program. It is intended as a simple template for VitralSDK application programmers. It features basic camera aids for JOGL applications. CameraExample CameraExample
