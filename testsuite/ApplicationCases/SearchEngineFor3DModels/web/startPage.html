VITRAL 3D MODEL SEARCH ENGINE

Vitral based search engine testbed

Welcome to the VITRAL 3D Model Search Engine page. This site is a web based search engine for 3D geometrical models, inspired in (almost cloned from!) the original Princeton's 3D Model Search Engine available at http://shape.cs.princeton.edu, as currently published in [FUNK2003] and [MIN2003].

Current differences between this search engine and original Princeton's search engine are:

  • Princeton's search engine is a lot bigger than this site, with more traffic and many more supported models, as VITRAL implementation doesn't have a web crawler yet. If you want to search for 3D models, use Princeton's original page.
  • Princeton's search engine is the original. This page is just an independent clone. If you want to reference this idea, please give credit to original authors at Princeton.
  • Absolutely all of the source code is free software, available in the net and distributed inside an easily reproducible project, included inside the Vitral SDK software distribution. Download page for Vitral SDK is at http://sophia.javeriana.edu.co/~ochavarr. Use VITRAL implementation if you are interested in studying current system (most posible if you are a computer science student, or other related academic person interested in pattern matching and shape matching).
  • This is an almost 99% java implementation of the search engine. Current implementation use java libraries (Vitral, JOGL), java applets and java servlets inside a TOMCAT web application container. Use VITRAL implementation if you like java programming language use in computer graphics applications. Currently, just some Fourier algorithms are used in C/C++, with JNI native code wrappers for SpharmonicKit and FTTW libraries.
  • This implementation integrates other shape matching techniques aside from those described in [FUNK2003] and [MIN2003]. Use this implementation if you want to compare different shape matching algorithms.
